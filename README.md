# How to Run Locally

- clone the repo on local machine
  `git clone https://github.com/sanketssc/dynamic-form`

- switch to project dir
  `cd dynamic-form`

- Install Dependencies
  `npm install`
  or
  `yarn`

- Run the project with
  `npm run dev`
  or
  `yarn dev`
